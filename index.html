STEP 01 // VirtualBox Installation
WHAT: Installed Oracle VirtualBox 7.2.6 on Windows 11 — our hypervisor software that will host and run all virtual machines. WHY: A hypervisor lets you run multiple operating systems on one physical machine simultaneously. Each VM thinks it's a real computer but it's isolated software. This mirrors how enterprise IT works at scale — companies run thousands of VMs on physical servers. NOTE: VirtualBox is a Type 2 (hosted) hypervisor — runs on top of Windows. Enterprise equivalent is VMware ESXi or Microsoft Hyper-V (Type 1 — runs directly on hardware).

STEP 02 // Windows Server 2022 ISO Download
WHAT: Downloaded the Windows Server 2022 180-day evaluation ISO directly from Microsoft's Evaluation Center. WHY: Windows Server 2022 is the OS that will run our Domain Controller. It's the current enterprise standard and what most IT environments are running. The ISO is the installation image — think of it as the digital version of a Windows install disc. NOTE: The 180-day eval is completely legitimate and widely used for lab and learning purposes. Microsoft provides it specifically for this reason.
