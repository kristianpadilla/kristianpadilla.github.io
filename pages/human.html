The SteamDeck Setup

// The Setup

My SteamDeck is a portable gaming device and a Linux machine that fits in my bag — both at the same time. I set up EmuDeck and turned it into a full retro emulation station on top of everything else, mostly focused on PS2. Growing up on PS2 games hit different, and being able to revisit them with upscaled graphics on that screen is something else.

EmuDeck is an emulator management tool that organizes and runs games from a huge range of classic systems. Getting everything configured — navigating the Linux file system in desktop mode, managing emulator settings, troubleshooting compatibility — is genuinely the same skill set as IT support. Just applied to something I actually enjoy doing on my own time.
